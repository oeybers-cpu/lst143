How We Protect Anonymity

  • No sign-in required - no email or personal data collected
  • Your chosen name is never sent to the server - only used locally
  • Server stores: timestamp, group, peer rated, score, and optional comment
  • All data is stored in a secure Google Sheet with restricted access
